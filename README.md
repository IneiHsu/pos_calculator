## English

### Introduction
**Smart POS Calculator** is a lightweight, mobile-first web application designed to turn your smartphone into a simple Point of Sale (POS) system. It combines a standard calculator with customizable preset buttons, making it ideal for small businesses, stalls, and quick sales events.

### Key Features
- **Customizable Setup:** Up to **5** quick-access product buttons with custom prices and colors  
- **Multi-language Support:** English / 中文 / 日本語  
- **Smart Counting:** Automatically tracks item quantities (e.g., tap **Red** twice = 2 items)  
- **Quick Multipliers:** Dedicated **x1.1** button (tax/service charge)  
- **Cash Register Logic:** Easy change calculation with **Cash / Received** button  
- **Sales History:** Stores transaction records locally (in your browser)  
- **CSV Export:** Download sales data as CSV for Excel/Google Sheets  
- **Offline Capable:** Works without internet once loaded

### How to Use

#### 1) Installation (iOS / Android)
1. Save `calculator.html` to your phone  
2. Open it in a browser (**Safari recommended on iOS**)  
3. Tap **Share** → **Add to Home Screen**  
4. Launch it from the home screen for the best full-screen experience

#### 2) Initial Setup
1. **Language:** Select your preferred language  
2. **Product Buttons:** Use the slider to choose **0–5** preset buttons  
3. **Customize:**
   - Enter the **Price** for each button  
   - Tap the **color circle** to open the color picker (**10 options**)  
4. Tap **Start** to enter the calculator view

#### 3) Making a Sale
- **Add Items:** Tap preset buttons to add items  
  - Example: Tap **[Red]** then **[Blue]**  
- **Multi-quantity:** Tap **[Red] → × → 3 → +** to add 3 Red items  
- **Basic Math:** Use standard keys (**+ / - / ×**) for adjustments  
- **Tax/Service:** Tap **x1.1** to multiply the current total by 1.1

#### 4) Payment & Change
1. When the total is ready, tap **Cash / Received** (green)  
2. Enter the amount received (e.g., `2000`)  
3. Tap **=** (blue)  
4. The screen shows the **Change due**

#### 5) Recording
- After the transaction, the bottom-right button becomes **Record** (black)  
- Tap **Record** to save and reset for the next customer

#### 6) History & Export
- Tap **History** (top-left) to view transactions  
- Tap **Download CSV** to export  
- Tap **Settings** (top-left icon) to edit prices/colors

## 中文

### 简介
**智能收银计算器**是一款轻量级的移动端网页应用，可将智能手机变成简易 POS 收银终端。它结合标准计算器与可自定义的商品快捷按键，适合小微商家、摊位与临时售卖场景。

### 主要功能
- **自定义设置：** 最多 **5** 个商品快捷按键，可设置价格与颜色  
- **多语言支持：** 中文 / English / 日本語  
- **智能计数：** 自动记录商品数量（例如点两次**红色**=2件）  
- **快捷倍率：** 专用 **x1.1** 按键（税费/服务费）  
- **收银逻辑：** 使用 **实收** 按键快速计算找零  
- **销售记录：** 浏览器本地保存交易历史  
- **CSV 导出：** 导出 CSV 以便 Excel/Google Sheets 管理  
- **离线使用：** 单文件加载后，无需网络即可运行

### 使用方法

#### 1) 安装（iOS / Android）
1. 将 `calculator.html` 保存到手机  
2. 用浏览器打开（iOS 推荐 **Safari**）  
3. 点击 **分享** → **添加到主屏幕**  
4. 从桌面启动，获得更佳的全屏体验

#### 2) 初始化设置
1. **语言：** 选择常用语言  
2. **商品数量：** 滑块选择 **0–5** 个预设按键  
3. **自定义：**
   - 输入每个按键对应的**价格**  
   - 点击左侧**颜色圆点**打开调色板（**10种颜色**）  
4. 点击 **开始使用** 进入电卓界面

#### 3) 销售流程
- **添加商品：** 点击彩色预设按键添加  
  - 示例：点 **[红色]** → 点 **[蓝色]**  
- **多件商品：** **[红色] → × → 3 → +**（添加 3 件红色）  
- **基础计算：** 可使用 **+ / - / ×** 手动调整  
- **税/费：** 点击 **x1.1** 将当前合计乘以 1.1

#### 4) 支付与找零
1. 总价确认后，点击绿色 **实收**  
2. 输入顾客支付金额（例如 `2000`）  
3. 点击 **=**（蓝色按键）  
4. 屏幕显示应找零金额

#### 5) 记录交易
- 交易完成后，右下角按键变为 **记录**（黑色）  
- 点击 **记录** 保存本次交易并重置界面

#### 6) 历史与导出
- 点击左上角 **查看记录** 浏览历史  
- 点击 **下载 CSV** 导出报表  
- 点击左上角 **设置**（齿轮）修改价格/颜色

## 日本語

### はじめに
**スマートレジ電卓**は、スマートフォンをシンプルなPOSレジに変える軽量Webアプリです。標準的な電卓機能に加えて、価格・色を設定できる商品ボタンを備え、個人商店・屋台・イベント販売に適しています。

### 主な機能
- **カスタマイズ設定：** 最大 **5** つの商品ボタン（価格・色を設定可能）  
- **多言語対応：** 日本語 / English / 中文  
- **スマート集計：** 個数を自動記録（例：**赤**を2回＝2個）  
- **クイック計算：** **x1.1** ボタン（消費税/サービス料）  
- **お釣り計算：** **預かり** ボタンで簡単に計算  
- **売上履歴：** ブラウザに取引履歴を保存  
- **CSVエクスポート：** CSVで出力しExcel等で管理  
- **オフライン動作：** 一度読み込めばネット接続なしで利用可能

### 使い方

#### 1) インストール（iOS / Android）
1. `calculator.html` をスマホに保存  
2. ブラウザで開く（iOSは **Safari推奨**）  
3. **共有** → **ホーム画面に追加**  
4. ホーム画面から起動すると全画面で快適に利用できます

#### 2) 初期設定
1. **言語：** 使用する言語を選択  
2. **商品ボタン数：** スライダーで **0〜5** を選択  
3. **カスタマイズ：**
   - 各ボタンの**価格**を入力  
   - 左側の**色の丸**をタップして色（**10色**）を選択  
4. **開始** をタップして電卓画面へ

#### 3) 販売フロー
- **商品追加：** 色付きの商品ボタンをタップ  
  - 例：**[赤]** → **[青]**  
- **複数個：** **[赤] → × → 3 → +**（赤を3個追加）  
- **基本計算：** **+ / - / ×** で手動調整可能  
- **税/手数料：** **x1.1** で合計を1.1倍

#### 4) 会計・お釣り
1. 合計が出たら緑色の **預かり** を押す  
2. 受け取った金額（例：`2000`）を入力  
3. **=**（青）を押す  
4. お釣りが表示されます

#### 5) 記録
- 会計後、右下のボタンが **記録**（黒）に変わります  
- **記録** を押すと保存され、次の会計のためにリセットされます

#### 6) 履歴と出力
- 左上の **履歴を見る** で過去の取引を確認  
- **CSV保存** でダウンロード  
- 左上の **設定**（歯車）で価格/色を変更
