Introduction

Smart POS Calculator is a lightweight, mobile-first web application designed to turn your smartphone into a simple Point of Sale (POS) system. It combines a standard calculator with customizable preset buttons, making it perfect for small businesses, stalls, or quick sales events.

Key Features

Customizable Setup: Set up to 5 quick-access product buttons with custom prices and colors.

Multi-language Support: English, Chinese, and Japanese.

Smart Calculation: Automatically tracks item quantities (e.g., tapping "Red" twice records 2 items).

Quick Multipliers: A dedicated x1.1 button for tax or service charge calculations.

Cash Register Logic: Calculate change easily with the "Cash" (Received) button.

Sales History: Local storage of transaction records.

CSV Export: Download your sales data as a CSV file for Excel/Google Sheets.

Offline Capable: Works entirely in your browser without an internet connection once loaded.

How to Use

1. Installation (iOS/Android)

Save the calculator.html file to your phone.

Open the file with a browser (Safari recommended for iOS).

Tap the "Share" button and select "Add to Home Screen".

Launch it like a native app for the best full-screen experience.

2. Initial Setup

Language: Select your preferred language.

Product Buttons: Use the slider to choose how many preset buttons you need (0-5).

Customize: * Enter the Price for each button.

Tap the colored circle to open the color picker and assign a specific color (10 options available).

Tap Start to enter the calculator view.

3. Making a Sale

Add Items: Tap the colored preset buttons to add items.

Example: Tap [Red] then [Blue].

Multi-quantity: Tap [Red], then ×, then 3, then + to add 3 Red items.

Basic Math: Use standard calculator keys (+, -, ×) for manual adjustments.

Tax/Service: Tap x1.1 to multiply the current total by 1.1.

4. Payment & Change

Once the total is ready, tap the green Cash button (or "Received").

Enter the amount given by the customer (e.g., 2000).

Tap = (blue button).

The screen will show the Change due.

5. Recording

After the transaction is complete, the bottom right button becomes Record (black button).

Tap it to save the transaction and reset the screen for the next customer.

6. History & Export

Tap History (top left) to view past transactions.

Tap Download CSV to save the report.

Tap the Settings icon (top left) to modify prices or colors.

简介

智能收银计算器 是一款轻量级的移动端网页应用，旨在将您的智能手机变成一个简单的销售终端（POS）系统。它结合了标准计算器和可自定义的预设按键，非常适合小微商家、摊位或临时售卖活动使用。

主要功能

自定义设置：最多设置 5 个快捷商品按键，可自定义价格和颜色。

多语言支持：支持中文、英文和日文。

智能计数：自动记录商品数量（例如：点击两次“红色”按键即记录为 2 件）。

快捷倍率：专用的 x1.1 按键，用于快速计算税费或服务费。

收银逻辑：使用“实收”按键轻松计算找零。

销售记录：本地存储交易历史。

CSV 导出：可将销售数据导出为 CSV 文件，方便在 Excel 中查看。

离线使用：单文件设计，加载后无需网络即可在浏览器中运行。

使用方法

1. 安装 (iOS/Android)

将 calculator.html 文件保存到手机。

使用浏览器打开（iOS 推荐使用 Safari）。

点击底部的“分享”按钮，选择 “添加到主屏幕”。

像原生 App 一样从桌面启动，获得最佳全屏体验。

2. 初始化设置

语言：选择您习惯的语言。

商品数量：滑动滑块选择需要的预设按键数量（0-5个）。

自定义：

输入每个按键对应的价格。

点击左侧的颜色圆点，弹出颜色选择面板，从 10 种颜色中选择一种。

点击 开始使用 进入计算器界面。

3. 销售流程

添加商品：点击彩色的预设按键添加商品。

示例：点 [红色] 然后点 [蓝色]。

多件商品：点 [红色] -> × -> 3 -> +，即可添加 3 件红色商品。

基础计算：可使用标准计算器按键进行手动调整。

税/费：点击 x1.1 将当前总额乘以 1.1。

4. 支付与找零

计算出总价后，点击绿色的 实收 按键。

输入顾客支付的金额（例如：2000）。

点击 =（蓝色按键）。

屏幕将显示需要找零的金额。

5. 记录交易

交易完成后，右下角按键会变为 记录（黑色按键）。

点击它保存当前交易，并重置屏幕准备接待下一位顾客。

6. 历史与导出

点击左上角的 查看记录 浏览历史交易。

点击 下载 CSV 导出报表。

点击左上角的 设置 图标（齿轮）可随时修改价格或颜色。

はじめに

スマートレジ電卓は、スマートフォンをシンプルなPOSレジに変える軽量なWebアプリです。標準的な電卓機能に加え、カスタマイズ可能な商品ボタンを搭載しており、個人商店、屋台、フリーマーケットなどに最適です。

主な機能

カスタマイズ設定：最大5つの商品ボタンを作成し、価格と色を自由に設定可能。

多言語対応：日本語、英語、中国語に対応。

スマート集計：商品の個数を自動的に記録（例：「赤」ボタンを2回押すと2個として記録）。

クイック計算：消費税やサービス料計算に便利な x1.1 ボタン。

お釣り計算：「預かり」ボタンを使って簡単にお釣りを計算。

売上履歴：取引履歴をブラウザ内に保存。

CSVエクスポート：売上データをCSVファイルとしてダウンロードし、Excelなどで管理可能。

オフライン動作：一度読み込めば、インターネット接続なしで利用可能。

使い方

1. インストール (iOS/Android)

calculator.html ファイルをスマホに保存します。

ブラウザで開きます（iOSの場合はSafari推奨）。

「共有」ボタンを押し、**「ホーム画面に追加」**を選択します。

ホーム画面からアプリのように起動すると、全画面で快適に使用できます。

2. 初期設定

言語：使用する言語を選択します。

商品ボタン数：スライダーでボタンの数（0〜5個）を決めます。

カスタマイズ：

各ボタンの価格を入力します。

左側の色の丸をタップするとカラーパレットが開き、10色の中から好きな色を選べます。

開始 をタップして電卓画面へ進みます。

3. 販売フロー

商品追加：色付きの商品ボタンをタップして追加します。

例：[赤] を押し、次に [青] を押す。

複数個：[赤] -> × -> 3 -> + の順に押すと、赤の商品を3個追加できます。

基本計算：通常の電卓キーも併用できます。

税/手数料：x1.1 を押すと、現在の合計額が1.1倍になります。

4. 会計・お釣り

合計金額が出たら、緑色の 預かり ボタンを押します。

お客様から受け取った金額（例：2000）を入力します。

=（青いボタン）を押します。

画面にお釣りが表示されます。

5. 記録

会計が終わると、右下のボタンが 記録（黒いボタン）に変わります。

これを押すと取引が保存され、画面がリセットされて次の会計の準備が整います。

6. 履歴と出力

左上の 履歴を見る をタップして過去の取引を確認できます。

CSV保存 をタップしてデータをダウンロードします。

左上の 設定 アイコン（歯車）から、いつでも価格や色を変更できます。
