### 简介
**智能收银计算器**是一款轻量级的移动端网页应用，可将智能手机变成简易 POS 收银终端。它结合标准计算器与可自定义的商品快捷按键，适合小微商家、摊位与临时售卖场景。

### 主要功能
- **自定义设置：** 最多 **5** 个商品快捷按键，可设置价格与颜色  
- **多语言支持：** 中文 / English / 日本語  
- **智能计数：** 自动记录商品数量（例如点两次**红色**=2件）  
- **快捷倍率：** 专用 **x1.1** 按键（税费/服务费）  
- **收银逻辑：** 使用 **实收** 按键快速计算找零  
- **销售记录：** 浏览器本地保存交易历史  
- **CSV 导出：** 导出 CSV 以便 Excel/Google Sheets 管理  
- **离线使用：** 单文件加载后，无需网络即可运行

### 使用方法

#### 1) 安装（iOS / Android）
1. 将 `calculator.html` 保存到手机  
2. 用浏览器打开（iOS 推荐 **Safari**）  
3. 点击 **分享** → **添加到主屏幕**  
4. 从桌面启动，获得更佳的全屏体验

#### 2) 初始化设置
1. **语言：** 选择常用语言  
2. **商品数量：** 滑块选择 **0–5** 个预设按键  
3. **自定义：**
   - 输入每个按键对应的**价格**  
   - 点击左侧**颜色圆点**打开调色板（**10种颜色**）  
4. 点击 **开始使用** 进入电卓界面

#### 3) 销售流程
- **添加商品：** 点击彩色预设按键添加  
  - 示例：点 **[红色]** → 点 **[蓝色]**  
- **多件商品：** **[红色] → × → 3 → +**（添加 3 件红色）  
- **基础计算：** 可使用 **+ / - / ×** 手动调整  
- **税/费：** 点击 **x1.1** 将当前合计乘以 1.1

#### 4) 支付与找零
1. 总价确认后，点击绿色 **实收**  
2. 输入顾客支付金额（例如 `2000`）  
3. 点击 **=**（蓝色按键）  
4. 屏幕显示应找零金额

#### 5) 记录交易
- 交易完成后，右下角按键变为 **记录**（黑色）  
- 点击 **记录** 保存本次交易并重置界面

#### 6) 历史与导出
- 点击左上角 **查看记录** 浏览历史  
- 点击 **下载 CSV** 导出报表  
- 点击左上角 **设置**（齿轮）修改价格/颜色
